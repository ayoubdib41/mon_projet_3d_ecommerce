# Mon Projet 3D E-commerce

## Instructions pour le PC Puissant :
1. Installer Python.
2. Installer Meshroom (AliceVision).
3. Ouvrir `backend/reconstruct_3d.py` et modifier la variable `MESHROOM_EXECUTABLE` avec le chemin de ton `meshroom_batch.exe`.
4. Dans le terminal, faire : `pip install -r requirements.txt`.
5. Lancer le serveur : `python backend/app.py`.
6. Ouvrir `frontend/index.html` dans le navigateur.

Note : Pour un vrai test, utilise au moins 20 photos d'un objet sous tous les angles.
